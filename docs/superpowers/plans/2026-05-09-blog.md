# Blog bilingue Astro — Plan d'implémentation

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ajouter un blog bilingue FR/EN au portfolio Astro de Benjamin Manguet avec une page liste (grille de cards) et des pages articles (contenu + sidebar TOC), accessibles depuis une nouvelle section "Articles" sur la homepage.

**Architecture:** Content Collections Astro avec dossiers `src/content/blog/fr/` et `src/content/blog/en/`. Routes `/blog/fr/[slug]` et `/blog/en/[slug]` générées statiquement. Homepage section `Articles.astro` qui affiche les 3 derniers articles dans la langue active via CSS `data-lang`.

**Tech Stack:** Astro 6.1.8, Content Collections (astro:content), TypeScript, Zod (inclus dans Astro), Vanilla JS pour l'IntersectionObserver du TOC.

const blog = defineCollection({

<script>
  // Highlight de la section active via IntersectionObserver
  const links = document.querySelectorAll<HTMLAnchorElement>('.toc__link');
  if (links.length > 0) {
    const headingIds = Array.from(links).map((l) => l.getAttribute('href')?.slice(1) ?? '');
    const headingEls = headingIds
      .map((id) => document.getElementById(id))
      .filter(Boolean) as HTMLElement[];

    const observer = new IntersectionObserver(
      (entries) => {
        entries.forEach((entry) => {
          const id = entry.target.id;
          const link = document.querySelector<HTMLAnchorElement>(`.toc__link[href="#${id}"]`);
          if (link) {
            link.classList.toggle('is-active', entry.isIntersecting);
          }
        });
      },
      { rootMargin: '0px 0px -70% 0px', threshold: 0 }
    );

    headingEls.forEach((el) => observer.observe(el));
  }
</script>
